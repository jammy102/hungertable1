
## 1. 프로젝트 제목 / 간단 설명

## 제목
배고픈 식탁(Hungry Table)

## 간단한 설명
저희가 만들고자 하는 프로젝트는 맛있는 레시피를 제공하는 페이지를 만드는 것입니다.
레시피를 작성하여 어떻게 요리하는지 과정을 상세히 사람들에게 제공합니다.
원하는 카테고리를 선택하여 좋아하는 요리를 찾아볼 수 있습니다.
또한 레시피에 대한 댓글을 남겨 평가를 남길 수 있습니다.
